
# [Astrotube](https://dino-tv.netlify.app/) | Video-Library App for Astroheads

A one place destination for all your Astro videos.

## Link

[astrotube.netlify.app](https://astrotube.netlify.app/)

## Demo

https://user-images.githubusercontent.com/67817330/179536803-9921376c-87f3-45c9-a247-aa4c0426da5b.mp4

## Features

- Light/Dark Theme Toggle
- User Authorization
    - Sign In
    - Sign Up
    - Sign Out
- Protected/Private Routes
- Video Listing
- Video Filter
- Video Player
- History
    - Watched videos automatically get added to the History
    - Remove Video from History
    - Clear all History
- Watch Later
    - Add videos to Watch Later
    - Remove videos from Watch Later
- Liked Videos
    - Liked Videos List
    - Remove videos from Liked Videos
- Playlists
    - Create Playlists
    - Delete Playlists
    - Add video to Playlists
    - Remove video from Playlists

## Tech Stack

**Client:** React, React Router, React Moment, Context API, React Player, [Astro UI](https://astro-ui.netlify.app/)

**Server:** Mockbee (Mock Backend)


## Run Locally

Clone the project

```bash
  git clone https://github.com/ark2002/Astrotube.git
```

Go to the project directory

```bash
  cd Astrotube
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
